# Corporate Landing Page 

A pixel-perfect, high-fidelity corporate landing page built from a professional Figma design. This project demonstrates advanced layout techniques using **React** and **Tailwind CSS**, focusing on fluid responsiveness from **320px to 1920px**.

## 🚀 Live Demo
[Insert your Link Here (https://comforting-lily-d23014.netlify.app/)]

## 🛠️ Tech Stack
* **Framework:** [React.js](https://reactjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)


## ✨ Key Features
* **Full Responsiveness:** Optimized for mobile (320px) up to ultra-wide screens (1920px).
* **Corporate UI:** Clean, professional aesthetic with a strict color palette (Blue `#003893` and Orange `#FF8000`).
* **Complex Hero Section:** Side-by-side layout with high-impact typography and CTA.
* **Hot Topics Ticker:** An interactive news grid with hover-state transitions.
* **Dynamic Stats Grid:** A 2x2 grid displaying firm achievements and global reach.
* **Insights Gallery:** A custom-weighted vertical image gallery using CSS Flexbox.
